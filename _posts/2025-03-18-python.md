## ¿Qué es Python?

Python es un lenguaje de programación de **alto nivel**, **interpretado** y que posee una filosofía que se centra en una **sintaxis** que permita un código **fácilmente legible** para el humano.

Python cuenta con una serie de ventajas en sus características: 

- **Multiplataforma**: Python permite ser descargado y usado en sistemas operativos Windows, Linux y macOS. 

- **Automatización y scripting**: Python es altamente usado en la creación de scripts y automatización de tareas debido a su sencilla legibilidad. 

- **Ciberseguridad y hacking ético**: Python es muy usado en este campo del hacking para la creación de exploits, automatización de ataques y análisis frente a amenazas.

- **Código abierto**: Python funciona bajo una licencia de código abierto, es decir, cualquiera puede contribuir al desarrollo del lenguaje y proponer mejoras para su continuo desarrollo, esta filosofía ha ayudado a Python a crecer y adaptarse rápidamente a los cambios en la industria.

## ¿Quién es el padre de Python?

Python fue lanzado por el programador neerlandés **Guido van Rossum** en 1991.

Van Rossum creó Python con la intención de que fuera un lenguaje sencillo, legible y fácil de usar y aprender. Aunque Python sea muy sencillo, no significa que no sea potente, de hecho Van Rossum lo creó para que fuera sencillo pero sin limitar la potencia y flexibilidad. 

## ¿Qué es el Zen de Python?

El zen de Python es un documento que establece los principios de la filosofía de Python, es como una guía que ayuda a los programadores a como se debería escribir el código. 

Cabe destacar que **NO** son **reglas estrictas** que haya que seguir siempre, pero reflejan la mentalidad que hace de Python un lenguaje sencillo y especial.

Algunos ejemplos de los aforismos que tiene el Zen de Python son: 

- Lo bello es mejor que lo feo
- Simple es mejor que complejo
- La legibilidad importa
