## ¿Qué es el string formatting?

El **formateo de cadenas** o string formatting en una forma en la que Python permite combinar texto junto con variables en una misma cadena.

## ¿Cuántas técnicas existen?

- **Operador** "**%**":

Se creó en el año 1991 en las primeras versiones de Python, además que se inspiró en el lenguaje **C**.

- **Método** "**.format()**":

Se creó en el año 2008 en las versiones de Python 2.6 y Python 3.0, se creó porque el operador anterior era más limitado para casos complejos.

- **F-Strings**:

Es la versión más reciente y recomendada para casi todo, se creó en el año 2017 a partir de Python 3.6 para hacer el formateo más rápido, claro y potente.
