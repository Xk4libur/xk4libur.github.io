Un conjunto (set) en Python es una colección de elementos que no están ordenados ni duplicados.

Los conjuntos se suelen usar en diferentes campos de la programación, ciberseguridad, automatización, etc... 

## ¿Cuáles son las características de los conceptos?

### · Unicidad

Los elementos no admiten elementos duplicados, esto los hace perfectos para situaciones donde se deben recolectar elementos únicos.

Ejemplo: 

![](/assets/images/python/112.png)

![](/assets/images/python/113.png)

La palabra **'rojo'** aparece una sola vez, aunque en el código aparece 2 veces.

### · Elementos desordenados

Los conjuntos no mantienen los elementos en un orden espećifico, es decir, no se almacenan en el orden en que se añaden, debido a que Python prioriza la velocidad en vez del orden.

![](/assets/images/python/116.png)

En este ejemplo se ejecutó el código **2 veces**, aunque en el primer resultado los elementos aparecen ordenados, puede ser que aparezcan desordenados, como ocurre con el segundo resultado.
