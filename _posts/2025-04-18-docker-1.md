Docker es una plataforma de contenedores que permite empaquetar, distribuir y ejecutar aplicaciones de forma aislada, rápida y portable.

El uso de contenedores es muy importante, ya que se pueden meter aplicaciones enteras junto con todas sus dependencias dentro de un contenedor para que se pueda ejecutar y mover de forma sencilla de una máquina a otra, asegurándose de que la aplicación funcionará correctamente independientemente del sistema en el que se ejecute la aplicación.

Docker tiene varias cosas positivas en cuanto al hacking, ya que si los desarrolladores meten sus aplicaciones web dentro de contenedores de Docker, es posible explotar vulnerabilidades reales dentro del contenedor sin afectar a la web real, además de que si un contenedor llega a corromperse o dañarse, es posible reconstruirlo desde 0 en segundos.
